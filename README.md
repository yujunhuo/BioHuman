# BioHuman project page

Run this folder with any static server, for example `python3 -m http.server 4173`, then open `http://localhost:4173`.

## Media placeholders

Put files at these exact paths; the page will detect and display them automatically:

- `public/media/teaser6.png` — main teaser image
- `public/media/pipeline完整视频.mov` — BioHuman pipeline video

H.264 MP4 is recommended. Replace the `SOON` dataset and code buttons in `src/main.js` after release URLs are available.
