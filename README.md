# BioHuman

Official project page for **BioHuman: Learning Biomechanical Human Representations from Video**.

BioHuman is an end-to-end framework that jointly predicts human motion and full-body muscle activations from monocular video. The project also introduces BioHuman10M, a large-scale dataset with synchronized video, motion, and biomechanical annotations.
